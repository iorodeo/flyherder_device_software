FlyHerderSerial.m  - matlab serial interface for controlling and communicating
with the flyherder device hardware.

For installation instructions INSTALL.txt in this directory.

For usage examples see flyherder_libs/matlab/flyherder_examples.

For help type "help FlyHerderSerial" from the Matlab command line.
